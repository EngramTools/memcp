## Decisions Made
- Trust levels calibrated: user/cli=0.8 (human-in-loop), auto-store=0.3 (noisy automated), import=0.4 (bulk unknown quality), default=0.5 (neutral)
- metadata is JSONB NOT NULL DEFAULT '{}' rather than nullable, simplifying all query code (no null checks)
- trust_history stored as JSONB array inside metadata field using jsonb_set/COALESCE append pattern, avoiding a separate audit table
- session_id and agent_role filters added to list() only (not search) per research anti-pattern guidance
