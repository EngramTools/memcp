**cli.rs** — Add `cmd_annotate` handler:

```rust
pub async fn cmd_annotate(
    store: &Arc<PostgresMemoryStore>,
    id: &str,
    tags_to_append: Option<Vec<String>>,
    tags_to_replace: Option<Vec<String>>,
    salience_input: Option<String>,
) -> Result<(), MemcpError>
```

Implementation:
1. `store.get(id).await?` — verify memory exists, get current state.
2. Capture `tags_before` from current memory tags (Vec<String> from JSONB).
3. **Tag mutation:**
   - If `tags_to_replace` is Some: use those directly as new tags.
   - Else if `tags_to_append` is Some: merge current tags + new tags, deduplicate via `HashSet`, sort.
   - Else: no tag change (tags_after = tags_before).
4. If tags changed: `store.update(id, UpdateMemory { tags: Some(new_tags), ..Default::default() }).await?`
5. **Salience mutation:**
   - Parse `salience_input`: if ends with 'x' → multiplier mode (parse f64 from prefix). Else → absolute mode (parse f64).
   - Read current salience: `store.get_salience_data(&[id.to_string()]).await?`. Get the entry or use defaults (stability=2.5, difficulty=0.3, count=0, last_reinforced=None).
   - Calculate `new_stability`: absolute → use value directly; multiplier → `(current.stability * multiplier).min(100.0)`.
   - Call `store.upsert_salience(id, new_stability, current.difficulty, current.reinforcement_count, current.last_reinforced_at).await?`
   - CRITICAL per Pitfall 1: Always pass current difficulty and reinforcement_count back to upsert — do NOT use defaults.
6. Compute diff:
   - `tags_added`: tags in new but not old.
   - `tags_removed`: tags in old but not new.
   - `salience_before`: current stability (before mutation). Only if salience was mutated.
   - `salience_after`: new stability. Only if salience was mutated.
7. Output diff JSON to stdout:
```json
{
  "id": "...",
  "changes": {
    "tags_added": ["tag1"],
    "tags_removed": [],
    "salience_before": 2.5,
    "salience_after": 3.75
  }
}
```
If no salience change, omit salience_before/salience_after. If no tag change, use empty arrays.
