## What Was Built

### Task 1: Temporal Extraction Module

New module `crates/memcp-core/src/pipeline/temporal/mod.rs` with:

- `EventTimePrecision` enum: `Decade`, `Year`, `Month`, `Day` with `as_str()` conversion
- `extract_event_time(content, birth_year, now) -> Option<(DateTime<Utc>, EventTimePrecision)>`
- 6 regex patterns compiled once via `std::sync::OnceLock`:
  1. Month-year: `in March 2020` → Month precision
  2. Absolute year: `in 2019` → Year precision (1900–2100 validated)
  3. Decade: `in the 90s` → Decade precision (century resolved to most recent past)
  4. Relative age: `when I was 6` → Year precision (requires birth_year from config)
  5. Relative month: `last March` → Month precision (most recent past occurrence)
  6. Relative day: `last Tuesday` → Day precision (most recent past occurrence)
- 15 unit tests covering all pattern types, edge cases (same-month, same-weekday, year out of range), and precedence

### Task 2: Workspace Scoping + CLI/MCP Integration

**postgres.rs:**
- `hybrid_search` gains `workspace: Option<&str>` param; post-filter: retain hits where `workspace == Some(ws) || workspace.is_none()`
- `hybrid_search_multi_tier` same post-filter pattern
- `hybrid_search_paged` forwards workspace to hybrid_search
- `recall_candidates` gains `workspace: Option<&str>`; dynamic SQL appends `AND (m.workspace = $5 OR m.workspace IS NULL)` when Some
- `recall_candidates_multi_tier` forwards workspace to delegates
- `list()` adds workspace condition to WHERE clause builder

**store/mod.rs:**
- `ListFilter` gains `workspace: Option<String>` field

**cli.rs:**
- `resolve_workspace(cli_flag, config) -> Option<String>`: CLI flag > `MEMCP_WORKSPACE` env > `config.workspace.default_workspace`
- `cmd_store` gains `workspace` param; calls `extract_event_time` and sets `event_time`/`event_time_precision` on `CreateMemory`
- `cmd_search`/`cmd_list`/`cmd_recall` gain `workspace` param; pass resolved workspace to store methods
- `format_memory_json` verbose branch includes `event_time`, `event_time_precision`, `workspace` always; compact branch includes them only when non-null

**main.rs:**
