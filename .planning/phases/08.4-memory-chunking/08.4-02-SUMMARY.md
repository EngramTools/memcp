## Decisions Made
- Used `unicode_sentences()` from unicode-segmentation for robust sentence detection (handles abbreviations like "Dr.", decimals like "3.14", URLs correctly)
- Overlap is sentence-count based: last N sentences of chunk A appear as first N sentences of chunk B
- Content below min threshold or fitting in a single chunk returns empty Vec -- no unnecessary chunking
- Topic extracted from first sentence of parent content, truncated to 60 chars with safe char boundary handling
- Added `floor_char_boundary` helper since the nightly `str::floor_char_boundary()` is not available on stable Rust
